Bem-vindo ao catálogo de jogos-teste!

Esta é a página inicial. Você poderá escolher cinco jogos para jogar:

  • Sistema de Ingresso: Você pode comprar ingressos para um show, desde que tenha disponível.
  • Aluguel de Jogos: Você pode alugar (e devolver) três jogos de tabuleiro.
  • Carrinho de Compras: Entre uma lista de três produtos, é possível adicionar ou remover eles do carrinho, por ordem e quantidade indeterminada.
  • Sorteador de Números: Sorteia uma quantidade de números indefinida, sem repetir eles.
  • Jogo do Amigo Secreto: Algoritmo que cria automaticamente os grupos para um jogo de amigo secreto.
